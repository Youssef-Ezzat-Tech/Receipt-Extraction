# Receipt Data Extraction (Bounding Boxes)

## What This Is
Annotated dataset of 30 receipt images with bounding boxes for structured field extraction (OCR training).

## Fields
Merchant Name | Date | Total | Subtotal | Tax | Payment Method | Address

## Quality
- 180 fields annotated (≈6 per receipt)
- ≥90% accuracy standard applied
- Edge cases handled (low quality, missing fields, format variation)

## Files
- `Exports/receipt_extraction_export.json` — bounding box annotations (JSON)  
- `docs` — labeling guidelines + project summary  
- `screenshots` — annotation proof  

## Skills Shown
Bounding box annotation • OCR data prep • multi-field extraction • handling messy real-world data
